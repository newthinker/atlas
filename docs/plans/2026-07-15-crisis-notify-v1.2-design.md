## 修订（R7）：「仍异常从句」机制化

1. **语义句表**：`CRISIS→WATCH` 删固定从句，改为：

```
情绪层连续 %d 个交易日回落至绿。危机状态退出，转入观察期。
```

2. **renderTransition 降级分支**：复用 ✅/🔽 的同一判定（`splitZones` 异常区非空）——非空时在语义句后追加从句：

```
其余层面仍有异常，见下。
```

防御路径：语义句为空（不可达转移）而异常区非空时，从句独立成段。

## 语义

- 三处自洽共用同一判定：符号（✅/🔽）、从句（无/有）、正文标题（7 指标全绿：/仍异常：）——组合矛盾在机制上不可能。
- 从句对**所有**含异常降级生效：CRISIS→WATCH、BREWING→WATCH、WATCH→NORMAL（当日可有非触发 AMBER，生产可达）在预览截断下获得同等保护。
- 措辞由「可能仍异常」改为「仍有异常」：从句仅在异常区确实非空时出现，确定性表述准确且更强。


含异常降级（BREWING→WATCH 同形态）：
```

情绪层连续 10 个交易日回落至绿。危机状态退出，转入观察期。其余层面仍有异常，见下。

仍异常：
🟡 …
```

## 测试要点

1. 全绿降级：NotContains「仍有异常」、NotContains「可能仍异常」（旧从句负向）。
2. 含异常降级：CRISIS→WATCH 与 BREWING→WATCH 各一条 Contains「其余层面仍有异常，见下。」。
3. 从句与 🔽 共享判定：对该判定做变异（非空→恒真/恒假）应同时击穿符号断言与从句断言（双拦）。
